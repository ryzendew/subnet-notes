Complete CSS Guide for Beginners

Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Why Use CSS?

  • Separation of content and presentation
  • Consistent styling across multiple pages
  • Faster page loading
  • Easier maintenance
  • Advanced control over layout

How to Add CSS to HTML

There are three ways to add CSS to HTML:

/* 1. External CSS (Recommended) */ <link rel="stylesheet" href="styles.css"> /* 2. Internal CSS */ <style> body { background-color: lightblue; } </style> /* 3. Inline CSS */ <h1 style="color:blue;">Heading</h1>

CSS Syntax

CSS consists of style rules that are interpreted by the browser and applied to the corresponding elements.

selector { property: value; property: value; }

CSS Rule Breakdown

  • Selector: Points to the HTML element you want to style
  • Property: The style attribute you want to change
  • Value: The value you want to assign to the property
/* Example */ h1 { color: navy; font-size: 24px; text-align: center; }

CSS Selectors

Selectors are used to select the HTML elements you want to style.

Basic Selectors

/* Element Selector */ p { color: red; } /* Class Selector */ .my-class { font-weight: bold; } /* ID Selector */ #my-id { background: yellow; } /* Universal Selector */ * { margin: 0; padding: 0; }

Combination Selectors

/* Descendant Selector */ div p { color: blue; } /* Child Selector */ div > p { font-size: 18px; } /* Adjacent Sibling Selector */ h1 + p { margin-top: 0; } /* General Sibling Selector */ h1 ~ p { color: green; }

Attribute Selectors

/* Attribute exists */ a[target] { color: red; } /* Attribute equals value */ input[type="text"] { border: 1px solid #ccc; } /* Attribute contains value */ a[href*="example"] { color: blue; }

Pseudo-classes and Pseudo-elements

/* Pseudo-classes */ a:hover { color: red; } li:first-child { font-weight: bold; } input:focus { border-color: blue; } /* Pseudo-elements */ p::first-line { font-weight: bold; } p::before { content: ">> "; } p::after { content: " <<"; }

CSS Box Model

Every element in CSS is represented as a rectangular box. The box model describes how the element's content, padding, border, and margin interact.

Box Model Interactive Example

Adjust the values to see how they affect the element:

Content Area
.box { width: 200px; /* Content width */ height: 100px; /* Content height */ padding: 20px; /* Space inside the border */ border: 5px solid black; /* Border around the element */ margin: 15px; /* Space outside the border */ /* Total width = width + padding-left + padding-right + border-left + border-right */ /* Total height = height + padding-top + padding-bottom + border-top + border-bottom */ }

Box-Sizing Property

The box-sizing property controls how the total width and height of an element is calculated.

.content-box { box-sizing: content-box; /* Default: width/height = content only */ } .border-box { box-sizing: border-box; /* width/height = content + padding + border */ }

Layout & Display

The display property determines how an element is displayed on the page.

Display Values

Positioning

The position property specifies how an element is positioned in the document.

.static { position: static; /* Default, follows normal document flow */ } .relative { position: relative; /* Positioned relative to its normal position */ top: 10px; left: 20px; } .absolute { position: absolute; /* Positioned relative to nearest positioned ancestor */ top: 0; right: 0; } .fixed { position: fixed; /* Positioned relative to viewport */ bottom: 20px; right: 20px; } .sticky { position: sticky; /* Toggles between relative and fixed based on scroll */ top: 0; }

Flexbox Layout

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

Flex Container Properties

.flex-container { display: flex; flex-direction: row; /* Direction: row, row-reverse, column, column-reverse */ justify-content: center; /* Horizontal alignment: flex-start, center, flex-end, space-between, space-around */ align-items: center; /* Vertical alignment: stretch, flex-start, center, flex-end, baseline */ flex-wrap: wrap; /* Wrapping: nowrap, wrap, wrap-reverse */ gap: 10px; /* Space between items */ }

Flex Item Properties

.flex-item { flex-grow: 1; /* Ability to grow if needed */ flex-shrink: 1; /* Ability to shrink if needed */ flex-basis: 100px; /* Default size before growing/shrinking */ align-self: center; /* Override container's align-items */ order: 2; /* Order of appearance */ }

CSS Grid Layout

CSS Grid is a two-dimensional layout system for the web.

Grid Container Properties

.grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr; /* Column sizes */ grid-template-rows: 100px 200px; /* Row sizes */ gap: 10px; /* Gap between grid items */ justify-items: center; /* Horizontal alignment inside grid cells */ align-items: center; /* Vertical alignment inside grid cells */ }

Grid Item Properties

.grid-item { grid-column: 1 / 3; /* Item spans from column line 1 to 3 */ grid-row: 1; /* Item placed in row 1 */ justify-self: start; /* Horizontal alignment for individual item */ align-self: end; /* Vertical alignment for individual item */ }

Responsive Design

Responsive design ensures your website looks good on all devices.

Media Queries

/* Mobile First Approach */ .container { width: 100%; padding: 10px; } /* Tablet */ @media (min-width: 768px) { .container { width: 750px; margin: 0 auto; } } /* Desktop */ @media (min-width: 992px) { .container { width: 970px; } } /* Large Desktop */ @media (min-width: 1200px) { .container { width: 1170px; } }

Responsive Units

.responsive-example { width: 100%; /* Percentage of parent */ max-width: 1200px; /* Maximum width */ padding: 2rem; /* Relative to root font size */ font-size: 1.2em; /* Relative to parent font size */ margin: 0 auto; /* Centering */ /* Viewport units */ height: 50vh; /* 50% of viewport height */ width: 80vw; /* 80% of viewport width */ }

Transitions & Animations

CSS allows you to create smooth transitions and animations.

Transitions

.transition-example { background: #3498db; transition: all 0.3s ease; /* property duration timing-function delay */ } .transition-example:hover { background: #e74c3c; transform: scale(1.1); }

Animations

/* Define animation */ @keyframes slide-in { 0% { transform: translateX(-100%); } 100% { transform: translateX(0); } } /* Apply animation */ .animated-element { animation: slide-in 1s ease forwards; }

CSS Best Practices

  • Use external stylesheets for better maintainability
  • Organize your CSS with comments and logical sections
  • Use semantic class names that describe purpose, not appearance
  • Follow a consistent naming convention (BEM, OOCSS, etc.)
  • Use relative units (em, rem, %) for better responsiveness
  • Minify CSS for production to improve loading times
  • Use CSS variables for consistent theming
  • Test your designs on multiple browsers and devices
